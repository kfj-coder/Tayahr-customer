# 🚗 Tayahr Customer

## 📱 Overview
Tayahr is a ride-hailing mobile application designed for the Comoros Islands, connecting passengers with drivers for fast and reliable transportation.

The client app allows users to request rides, track drivers in real-time, and manage their trips seamlessly.

The application is available on both Android and iOS platforms.

---

## 🚀 Features
- 📍 Real-time location & ride request
- 🚗 Driver matching system
- 🗺️ Live ride tracking
- 📜 Trip history
- 🔔 Notifications
- ⭐ Driver rating system
